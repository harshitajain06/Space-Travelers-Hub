### Prerequisites

In order to run this project you need:
- Webpack Setup
- Visual Studio Code

### Setup

Clone this repository to your desired folder:
Example command:

```sh
 cd ReactGroupProject
 git clone https://github.com/harshitajain06/ReactGroupProject.git
```
